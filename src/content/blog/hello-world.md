This is the first post on ~/home/ash.

## Why I Built This

I wanted a simple, fast, personal corner of the internet. No frameworks shipping megabytes of JavaScript. No CMS with a login page. Just Markdown files, a static site generator, and an Unraid server doing the heavy lifting.

## The Stack

- **Astro** for static site generation
- **JetBrains Mono** for that terminal feel
- **SWAG** on Unraid as the reverse proxy
- **Git pull + build** via User Scripts for deployment
